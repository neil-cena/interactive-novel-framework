## App name

**The Cellar Debt**

## Short description (e.g. 80 chars for Play)

Interactive story with choices and D&D-style combat. Your decisions shape the outcome.

## Full description

The Cellar Debt is an interactive novel where your choices matter. Navigate branching storylines, manage inventory, and face turn-based combat. Save your progress locally or in the cloud and revisit different paths.

Features:
- Branching narrative with multiple endings
- Turn-based combat
- Inventory and equipment
- Multiple save slots with optional cloud sync
- Play in the browser as a PWA or install the native app
